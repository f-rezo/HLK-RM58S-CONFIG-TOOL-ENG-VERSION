# HLK-RM58S-CONFIG-TOOL-ENG-VERSION
# hlk-rm58s configuration tool english version
# using:
# 1 choose com port
# 2 pull down gpio0 to gnd for 1s to enter config mode(at mode)
# 3 click search to search module
# 4 click read to read parameter
# 5 modify parameter as u want
# 6 click write to write parameter to rm58s
# 7 rm58s will auto reboot to new config and exit config mode. so if u want to reconfig again, repeat from step 1
